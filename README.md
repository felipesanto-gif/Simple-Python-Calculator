# Simple-Python-Calculator
Este projeto apresenta uma calculadora simples em Python, desenvolvida para praticar conceitos básicos como entrada de dados, variáveis e operações matemáticas. Realiza as quatro operações fundamentais e exibe os resultados no terminal, servindo como base para automações e soluções mais complexas.

num1 = float(input("digite o primeiro número: "))
num2 = float(input("digite o segundo número: "))

soma = num1 + num2
subtracao = num1 - num2
multiplicacao = num1 * num2
divisao = num1 / num2

print("Soma:", soma)
print("Subtração:", subtracao)
print("Multiplicação:", multiplicacao)
print("Divisão:", divisao)
